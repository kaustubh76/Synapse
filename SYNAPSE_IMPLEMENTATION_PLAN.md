# EXECUTIVE SUMMARY

**Synapse** is a decentralized intent propagation network where AI agents broadcast needs and competing service providers bid to fulfill them. Instead of point-to-point API calls, agents emit "intents" that propagate through a mesh network, creating a self-organizing marketplace for AI services.

**The Core Innovation**: Replace "Agent calls Server" with "Agent broadcasts Intent â†’ Network fulfills it"

**Tech Stack**:
- **Crossmint**: Agent wallets, payment escrow, multi-chain settlement
- **Eigencloud**: Verifiable compute (EigenCompute), agent identity (ERC-8004), TEE attestation
- **x402**: HTTP-native micropayments for intent settlement
- **thirdweb**: Multi-chain facilitator (80+ chains, 4000+ tokens)

â€¢ PriceScore = (MaxBudget - BidPrice) / MaxBudget
â€¢ ReputationScore = ProviderRep / 5.0
â€¢ TEE_Bonus = 1.2 if TEE attested, else 1.0

Default weights: w1=0.4, w2=0.4, w3=0.2

Example:
MaxBudget = $0.02, Bid = $0.008, Reputation = 4.8, TEE = true
PriceScore = (0.02 - 0.008) / 0.02 = 0.6
ReputationScore = 4.8 / 5.0 = 0.96
TEE_Bonus = 1.2

Score = 0.4 Ã— 0.6 + 0.4 Ã— 0.96 + 0.2 Ã— 1.2 = 0.864
